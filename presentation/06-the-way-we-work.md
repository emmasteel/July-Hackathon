# The way we work

No matter the project, we move through the same phases. They help us build real understanding, build with confidence, and verify before we ship.

![Diagram: six pixel-style boxes in a row — Explore, Specify, Plan, Implement, Review, Learn — with forward arrows between them and curved arrows looping back, showing the journey is iterative and messy. Explore is terracotta, the four Create steps are purple, Learn is green.]({{ '/assets/img/loop.png' | relative_url }})

<ul class="cards">
  <li><span class="tag" style="background:var(--explore)">Explore</span><p>Find what's possible, then pick a path worth taking. Go wide before going deep.</p></li>
  <li><span class="tag" style="background:var(--create)">Create</span><p>Turn the chosen direction into something real: <strong>specify → plan → implement → review</strong>.</p></li>
  <li><span class="tag" style="background:var(--survive)">Survive</span><p>Learn so our future selves start ahead — and have fun along the way.</p></li>
</ul>

<p class="pixel">The real journey is messy — you can always step back a phase. That's OK.</p>
