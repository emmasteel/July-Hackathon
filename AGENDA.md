## How to pace yourselves

- **Protect the spine.** If you're running short on time, a working C1–C5 beats a half-done
  stretch goal. Focus there first.
- **Split the work by role.** BAs can lead C2 while developers set up C1/C3; testers prep
  C4/C5; security folks scope S1. See the role table in the [README](README.md).
- **Buffer built in.** Each block has slack. If you finish early, start a stretch goal.
- **Ask for help.** Facilitators are circulating. A stuck team should raise a hand rather
  than burn 15 minutes.
